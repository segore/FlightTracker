OpenSky Einstellungen

Ein Cloudflare Worker als Proxy wird benötigt (kostenlos, ~5 min Setup). OpenSky Credentials werden im Worker als Secrets gespeichert, nicht im Browser. Anleitung: cloudflare-worker/worker.js in deinem Cloudflare-Dashboard deployen.

@if (openSky.hasWorker()) {
✅ Worker konfiguriert: {{ openSky.workerUrl() }}
}

Tipp: In Cloudflare unter Worker → Settings → Variables and Secrets die Secrets OPENSKY_CLIENT_ID und OPENSKY_CLIENT_SECRET anlegen.
